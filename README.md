### Hi there 👋

I'm a bioinformatician at **UCL**, working on omics pipeline development in **evolution and ecology**.
I build **reproducible, FAIR data pipelines** — mostly [Nextflow](https://www.nextflow.io/) / nf-core
workflows for high-throughput sequencing data — and care a lot about making science others can re-run.

- 🧬 Background in phylogenetics, molecular evolution, and alternative splicing regulation
- 🔬 Hands-on with RNA-Seq, MS-Proteomics, and Hi-C across evo-devo & ecology questions
- 🛠️ Nextflow / nf-core pipelines · Unix · R · Perl · Python
- 🐝 PhD at the CRG (Barcelona); postdoc at UCL on wasp caste evolution; pipeline work at Seqera Labs

### 🌿 Eco-Flow

I built **[Eco-Flow](https://eco-flow.github.io/)** — a BBSRC-funded community developing
open-source Nextflow pipelines for **agri-ecology**, aiming to close *"the omics gap in ecology"*
with reproducible, accessible and scalable workflows (built to nf-core standards, tested with nf-test).

- 🌐 Website: [eco-flow.github.io](https://eco-flow.github.io/)
